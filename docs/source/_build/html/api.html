basicsynbio.bsb_io.import_ice_parts(ice_user_config: dict, *ice_nums, ice_root='https://public-registry.jbei.org/', file_type='original', format='genbank')[source]¶
Returns

a BasicPart object using an entry on a JBEI-ICE instance.

Uses icebreaker under the hood. Refer to icebreaker documentation <https://edinburgh-genome-foundry.github.io/icebreaker/> for further information.

Parameters

  • ice_user_config (dict) – Either {email: password:} or {token: client:}. Note, compared to icebreaker, ‘Root’ is a separate default argument (ice_root).

  • *ice_nums – Part ID numbers as strings e.g. “17338”. Note, the distinction between Part ID and Part ID number. For instance, number is 17338 for ID=JPUB_017338.

  • ice_root (string) – Root of ice registry.

  • file_type (string) – The file type to download e.g. “original”, “genbank” or “fasta”.

  • format (string) – Format of the downloaded file. If file_type == “fasta”, format must also be “fasta”.

def default(self, o):
    try:
        iterable = iter(o)
    except TypeError:
        pass
    else:
